## 用户注册
#### 调用
* username:用户名
* password:密码
```
gstore_user_register(username, password)
```
#### 返回
```python
{
  "status": "OK",  # "OK","FAIL",
  "msg":"注册成功", # 是否成功，失败原因
  "result":[
    "userid":"dadasdwqed"
  ]
}
```

## 用户发过的微博
#### 调用
* username:用户名
* offset:开始位置
* size:数量，用于分页
```
gstore_user_weibo(username, offset, size)
```
#### 返回
该用户发过的微博，时间倒序
```python
{
  "status": "OK",  # "OK","FAIL",
  "msg":"查询成功", # 是否成功，失败原因
  "result":[
    {
      "username":"usernname", # 该微博用户名称
      "content":"这是一条微博", # 微博内容
      "post_time":"2019-01-01 11:11:11"， #发送时间
    }
  ]
}
```
